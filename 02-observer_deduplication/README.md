# Observer deduplication [![status](http://assets.dull.systems:8080/status?id=scaling_shiny/02-observer_deduplication/README.md)](https://www.dull.systems/observer-deduplication "")

# Goal
Automate the removal of dynamically created Shiny observers.

# Summary
The issue is discussed at length in [this companion post](https://www.dull.systems/observer-deduplication).

`observer_dedup.R` provides a function that:
 - Evaluates an expression, identified by a unique ID, under a reactive domain.
 - Keeps track of the shiny observers generated by that expression.
 - Destroys those observers when that same expression is reinvoked.

# Requirements:
- shiny ≥ 0.13.0
